The Fyrebox Shortcode plugin allows you to integrate a game from Fyrebox into your Wordpress Blog by using a Wordpress shortcode.

== Description ==

The Fyrebox Shortcode plugin allows you to easily integrate a html5 game widget that you have built on Fyrebox.co into your Wordpress Blog by using a Wordpress shortcode.
Use it like that in your blog post: `[fyrebox gid=GAME_ID gt=GAME_TYPE /]`. The shortcode for your game is located in the "publish" page of your page that you can access on the dashboard

Examples:

`[fyrebox gid="i6WOeUCiUl" gt="8" /]`
Embed the  SoundCloud trivia shown on fyrebox.co

`[fyrebox gid="mlEZhG7KpF" gt="5" /]'
Embed the word search puzzle shown on fyrebox.co

== Installation ==
Just copy the folder fyrebox-shortcode folder in the folder wp-content/plugin of your page and activate it on the plugin list.
